SQL Challenge


SELECT
    mov_title AS "Movie Title"
    mov_year AS Year  "Year Released"
    mov_time AS "Running Time"
    mov_lang AS "Language"
    rating.rev_stars AS "Rating"
FROM movie
JOIN rating ON rating.mov_id = movie.mov_id
WHERE rating.rev_stars > 8
AND
    mov_year BETWEEN 1950 AND 2000
AND
    mov_time > 120
ORDER BY mov_year ASC

This challenge was to test my SQL skills by querying a movies database using a sub-query

My query retrieves all films with a rating of more than 8, that where made between 1950 and 2000 and running time of over 2 hours.

To do this I joined the the movie table with the rating table using their matching primary key.

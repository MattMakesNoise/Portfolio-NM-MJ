Homepage Reflection

@mixin tooltip-main {

background-color: $dark-blue;

color: $white;

position: absolute;

display: none;

max-width: 260px;

text-align: center;

}


.show {

.tooltip {

@include tooltip-main;

padding: 20px;

p {

margin-top: 1rem;

font-size: 0.875rem;

font-weight: $light;

}

&::after { //Creates the dropdown arrow

content: '';

border-left: solid transparent 30px;

border-right: solid transparent 30px;

border-top: solid $dark-blue 30px;

bottom: -29px; //Makes it sit flush with the bottom of the element

height: 0;

left: 37.5%; //Centers the arrow

position: absolute; //Makes it appear below

margin-left: 0px;

}

}

&:hover .tooltip {

display: block;

pointer-events: auto;

}

}

This is an example of using SASS and CSS to create a tooltip that appears when you hover over a clients logo.

The top section is an example of a mixin, which is a re-useable block of code which helps to reduce time.

There are also examples of variables, in this case for colors used within the site. This is helpful because if at any point there is to be a change to a sites color scheme, the variables can be changed rather than every instance of the color being implemented.

The second block contains an example of using psuedo elements, in this case using transparent borders to create an arrow shape that protrudes out of the elements container.

It also demonstrates the use of nesting code within SASS and commenting code which is useful when you come back to a project and have no idea what anything does anymore!
